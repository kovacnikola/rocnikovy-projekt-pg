## Spuštění

1. Vytvořte lokální kopii repozitáře:
    > git clone https://github.com/kovacnikola/rocnikovy-projekt-pg.git
2. Nainstalujte závislosti:
    > yarn install
3. Spusťte prostředí:
    > yarn start
